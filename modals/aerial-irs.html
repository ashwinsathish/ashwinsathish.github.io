Overview

This project presents a simulation and analysis of UAV-assisted communication systems utilizing Intelligent Reflecting Surfaces (IRS) to enhance signal quality. The study assesses system performance through outage probability across various transmit signal-to-noise ratios (SNR) and implements a robust framework for real-time phase adjustment at the IRS.

System Model

The system comprises three key components: the source (transmitter), the IRS (mounted on a UAV), and the destination (receiver). The implementation incorporates:

  • Path-loss Model: Using path-loss exponent of 2.7 for urban environment
  • Nakagami-m Fading: For modeling multipath scattering effects
  • Inverse Gamma Shadowing: To represent large-scale variations in received signal power

Performance Analysis

Analysis of outage probability vs transmit SNR for three different spectral efficiency threshold values (1, 2, and 3 b/s/Hz). Higher spectral efficiency thresholds result in increased outage probabilities at all SNR levels. Optimal phase shift simulations show improved performance compared to standard simulations.

At a fixed transmit SNR of 10 dB, outage probability increases linearly with spectral efficiency threshold. This trend is consistent across standard simulations, optimal phase shift simulations, and analytical approximations.

UAV Swarm Modification

The original single UAV-IRS simulation code was modified to test multiple UAV scenarios through the following extensions:

  • Data Structure Modification: Individual channel matrices for G_Sr, G_rD (Nakagami fading) and L_Sr, L_rD (Inverse Gamma shadowing) were converted to cell arrays to handle multiple UAVs
  • Channel Summation: The end-to-end channel gain was modified to accumulate contributions from each UAV's IRS elements
  • Phase Configuration: The optimal phase shift computation loop was nested within a swarm loop to calculate phase adjustments for each UAV's reflecting elements
  • Statistical Analysis: The Laplace transform multiplication and CDF calculation were adjusted to process N*swarm_size elements instead of just N elements

Outage probability plots were generated for swarm configurations of 2 and 3 UAVs under the same simulation parameters.
