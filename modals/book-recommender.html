Overview

A recommendation system implementation using hyperbolic geometry for analyzing book recommendations. The project implements a content-based recommendation approach utilizing advanced geometric embedding techniques.

Approach

The project explores alternative embedding techniques for recommendation systems:

  • Analysis of embedding space representations
  • Investigation of geometric mapping techniques
  • Comparative approach to similarity measurement

System Model

Implementation details of the recommendation system:

  • Dataset: 115 books with descriptions
  • Embedding Model: Poincare Ball Model
  • Dimensionality: 2-dimensional embeddings
  • Feature Extraction: TF-IDF vectorization

Methodology

Key computational techniques and processing steps:

  • TF-IDF Vectorization of Book Descriptions
  • Graph Network Construction
  • Poincare Embedding Model Training
  • Distance-based Recommendation Generation

Recommendation Approach

Recommendation generation method:

  • Three recommendation types:
    • Genre-based recommendations
    • Description-based recommendations
    • Combined genre and description recommendations
  • Distance measure for similarity calculation
  • Top 10 most similar book recommendations
