Overview

An AI-powered fashion outfit recommender that combines sentence transformers for semantic search with GPT-2 for natural language outfit suggestions. The system offers personalized fashion recommendations through a conversational interface, allowing users to receive and refine outfit suggestions based on their preferences.

System Architecture

  • Semantic Search: Utilizes MiniLM-L6-v2 sentence transformer model for encoding product descriptions and finding semantically similar items
  • Natural Language Generation: Implements GPT-2 medium model for generating contextual outfit suggestions based on user queries
  • Product Embeddings: Creates combined text embeddings from multiple product attributes including name, type, group, department, and detailed descriptions
  • Interactive Feedback: Maintains memory of disliked items to refine future recommendations

Key Features

The system implements features across three priority levels:

  • High Priority:
    • Personalized outfit recommendations based on user preferences
    • Complete outfit coordination including accessories
    • Interactive feedback system
  • Medium Priority:
    • Occasion-based outfit suggestions
    • Regional preference customization
  • Lower Priority:
    • Outfit customization capabilities
    • Social media trend integration

Technical Implementation

  • Core Components:
    • SentenceTransformer with MiniLM for text embeddings
    • GPT-2 medium model for natural language outfit suggestions
    • PyTorch cosine similarity for matching
    • Top-k retrieval for recommendation ranking
  • Implementation Approach:
    • Data aggregation from HNM dataset
    • Sentence embeddings for semantic item matching
    • Interactive feedback loop for recommendation refinement
    • Visual representation through Matplotlib

Current Limitations

  • Dependency on static dataset limits reflection of latest trends
  • Computational overhead from dual model usage affecting response time
  • Binary feedback system limiting preference granularity
  • Name-based matching accuracy constraints in feedback processing
