Overview

A comprehensive virtualized 5G testbed implementation using Open5GS and UERANSIM, designed to support multiple use cases in next-generation mobile networks.

Technical Implementation

  • Core Network: Deployed Open5GS components including AMF, SMF, UPF, and NRF
  • RAN Simulation: Integrated UERANSIM for gNB and UE simulation
  • Network Slicing: Implemented dynamic slice configuration for different use cases
  • API Development: Created RESTful APIs for network management and monitoring

Key Features

  • eMBB Slice: High-bandwidth video streaming with QoS guarantees
  • URLLC Slice: Ultra-reliable communication with <2ms latency
  • mMTC Slice: Support for massive IoT device connectivity
  • Real-time Analytics: Network performance monitoring and visualization
