Overview

This project presents a spatial-aware IRS-assisted communication system that utilizes LiDAR technology for dynamic beamforming in vehicular networks. The implementation integrates LiDAR's precise spatial mapping capabilities with IRS phase shift optimization to enhance vehicular communications in dynamic urban environments where physical obstructions frequently disrupt signal paths.

System Model

The system implements a three-component architecture with direct and reflected signal paths. The base station uses a 4×2 antenna array positioned at (30, 1.5, 3), while a 256-element IRS array is co-located with the LiDAR at (0, 0, 2). Vehicle position is dynamically tracked at a fixed height of 1m. The channel model incorporates:

  • Direct path channel (Vehicle ↔ BS): h_d ∈ C^(M×1)
  • IRS-assisted path channels: H_r ∈ C^(M×N) (IRS to BS) and h_v ∈ C^(N×1) (Vehicle to IRS)
  • Combined MIMO channel incorporating Rician fading with optimized phase shifts

LiDAR Integration

LiDAR data processing follows the DeepSense 6G dataset specifications with a 16-meter scanning range, 10Hz frequency, and 111ms scanning resolution. Raw point cloud data is collected as D(t) ∈ R^(M×2), combining distance and angular measurements. The system performs temporal aggregation over N_t = 80 scans and implements a sophisticated blockage analysis pipeline that includes:

  • Moving average temporal analysis over 30 samples to identify blockage periods
  • Angular window analysis across 105° for precise position verification
  • Blockage signature creation through distance-angle matrix transformation
  • Real-time coordinate extraction for vehicle tracking and beamforming optimization

Performance Analysis

Performance evaluation demonstrates several key achievements in the system design. Rate analysis shows maximum achievable rates near the IRS position, with an average 10.5% improvement in achievable rates compared to the non-IRS model across all vehicle positions. Array configuration analysis reveals that 16×16 IRS arrays with 2×2 grouping outperform 8×8 full CSI configurations, demonstrating the effectiveness of element grouping strategies in larger arrays.

The element grouping study particularly shows how (k×k) grouping effectively reduces (M×M) arrays to (M/k×M/k) configurations while maintaining significant performance advantages. This finding has important implications for practical implementations where computational efficiency must be balanced with performance requirements.
